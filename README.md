# RepoTracker

The blurb

## Installation & usage

### Installation

* Clone or download the repo.
* Open terminal and navigate to `repotracker`
* Run `npm i` to install the dependencies.

### Usage

* run `npm run dev`
* In your browser you may now open the RepoTracker app at the url `localhost:8080`
* Enjoy.
